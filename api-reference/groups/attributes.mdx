All databases belong to a group. Each group has one primary location and can have multiple replica locations.

Locations can be added or removed to groups as needed.

#### Attributes

<ParamField path="name" type="string">
  The group name, **unique** across your organization.
</ParamField>

<ParamField path="uuid" type="string">
  The group universal unique identifier (UUID).
</ParamField>

<ParamField path="regions" type="string[]">
  An array of location codes the group is located.
</ParamField>

<ParamField path="primary" type="string">
  The primary region location code.
</ParamField>

<ParamField path="archived" type="boolean" default="false">
  A boolean representing whether the group is archived or not.
</ParamField>

<ResponseExample>

```json
{
  "archived": false,
  "locations": ["ams", "bos", "lhr"],
  "name": "default",
  "primary": "lhr",
  "uuid": "0a28102d-6906-11ee-8553-eaa7715aeaf2"
}
```

</ResponseExample>
