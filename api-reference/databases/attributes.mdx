All databases belong to a group. Turso automatically manages replicas of your database for each location in your [group](/api-reference/groups).

#### Attributes

<ParamField path="Name" type="string">
  The database name, **unique** across your organization.
</ParamField>

<ParamField path="DbId" type="string">
  The database universal unique identifier (UUID).
</ParamField>

<ParamField path="hostname" type="string">
  The DNS hostname used for client LibSQL and HTTP connections
</ParamField>

<ParamField path="IssuedCertLimit" type="number">
  Todo
</ParamField>

<ParamField path="IssuedCertCount" type="number">
  Todo
</ParamField>

<ParamField path="regions" type="string[]">
  An array of location codes the database.
</ParamField>

<ParamField path="primaryRegion" type="string">
  The primary region location code.
</ParamField>

<ParamField path="version" type="string">
  The current libSQL version this database is running.
</ParamField>

<ParamField path="group" type="string">
  The name of the group this database belongs to.
</ParamField>

<ParamField path="type" type="string" default="logical">
  The string representing the object type.
</ParamField>
