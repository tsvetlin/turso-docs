---
sidebarTitle: Update Version
openapi: "POST /v1/organizations/{organizationName}/databases/{databaseName}/update"
---

<Warning>

This operation causes some amount of downtime to occur during the update process. The version of libSQL server is taken from the latest built docker image.

</Warning>

<RequestExample>

```bash cURL
curl -L -X POST 'https://api.turso.tech/v1/organization/{organizationName}/databases/{databaseName}/update' \
-H 'Authorization: Bearer TOKEN'
```

```ts Node.js
import { createClient } from "@turso/api";

const turso = createClient({
  organization: "...",
  token: "",
});

const database = await turso.databases.update("my-db");
```

</RequestExample>
