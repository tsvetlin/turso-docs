<Info>

You cannot delete a personal organization, or organizations that have databases.

</Info>

<RequestExample>

```bash cURL
curl -L -X DELETE https://api.turso.tech/v1/organizations/{organizationName} \
  -H 'Authorization: Bearer TOKEN'
```

```ts Node.js
import { createClient } from "@turso/api";

const turso = createClient({
  organization: "...",
  token: "",
});

const response = await turso.organizations.delete("mycompany");
```

</RequestExample>
