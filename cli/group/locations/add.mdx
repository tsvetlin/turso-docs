You can replicate databases globally by adding locations to a group using the following command:

```bash
turso group locations add <group-name> <location-code> [flags]
```

## Flags

| Flag           | Description                                |
| -------------- | ------------------------------------------ |
| `-w`, `--wait` | Wait for group to be ready before exiting. |
